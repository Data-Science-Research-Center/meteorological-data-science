About this app

The objective of this system is to demonstrate the application of the statistical programming language R in computer solutions applied to time series statistics. In addition to this web system, the project includes project management, file manipulation, and results experimentation. The project uses a modular structure that benefits the understanding and scalability of new functionalities that can be added later.

App

The web system is written using the R Shiny web development framework and demonstrates the use of HTML templates to customize and stylize the user interface. The app is deployed through RStudio's webservice shinyapps.io . Additionally, the project is uploaded in the remote versioning service GitHub
